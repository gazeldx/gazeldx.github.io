## 设置VIM
### Mac下
```shell
$ touch ~/.vimrc
$ vi ~/.vimrc # 加入以下内容:
filetype on
syntax on
```
这样就有了高亮了.

## VIM常用的命令
```shell
yy # 拷贝本行
p # 在本行后粘贴 P 是在本行前粘贴
e # Go next word
b # Go previous word
^ # Go to start
$ # Go to end
```
